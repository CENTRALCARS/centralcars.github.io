Gestion des Dossiers Clients

Important - Mise à jour des données sur GitHub :

Pour que vos modifications soient prises en compte sur https://centralcars.github.io/dossiers_clients.json, vous devez :

  1. Cliquer sur "Télécharger JSON".
  2. Remplacer le fichier dossiers_clients.json existant dans votre dépôt GitHub par celui que vous venez de télécharger.
  3. Effectuer un "commit" et un "push" de vos modifications sur GitHub.

La sauvegarde directe vers cette URL depuis cette page n'est pas possible.
